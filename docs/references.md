# RIGRUNNER — Visual References

A scrapbook of **reference art / inspiration images** for the look of the game and its assets.
This is the *visual* companion to [`ideas.md`](ideas.md) (text brainstorms) — drop images that
capture a shape language, palette, mood, or specific asset idea we might want to borrow from.

> ⚠️ **Not committed direction.** Like `ideas.md`, this is raw inspiration, not a decision. An
> image here means "worth looking at", not "we're building this".

## How to use

1. Drop the image file into [`references/`](references/) (PNG/JPG; keep filenames `kebab-case`).
2. Add an entry below: the image, where it's from (if known), and **one line on what to borrow** —
   the silhouette? the palette? the way it shows wear? Be specific about the *takeaway*, since the
   image alone doesn't say why it's here.

When a reference actually shapes a built asset, note that asset's name in its entry so we can trace
"what inspired this".
